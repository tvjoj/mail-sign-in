Základný postup

  1. Otvorte Outlook v prehliadači cez outlook.office.com.
  2. Prihláste sa údajmi, ktoré ste dostali.
  3. Dokončite 2FA podľa priloženého PDF návodu cez aplikáciu Microsoft Authenticator.
  4. Ak potrebujete pridať zdieľaný priečinok, použite druhý PDF návod.

Nasadenie na GitHub Pages

  1. V GitHube vytvorte nový repozitár, napríklad mail-navody.
  2. Nahrajte obsah priečinka tak, aby bol súbor index.html v koreňovom adresári a PDF/TXT boli v priečinku assets.
  3. V repozitári otvorte Settings → Pages.
  4. V časti Build and deployment nastavte Deploy from a branch, vyberte vetvu main a priečinok /root.
  5. Po uložení bude web dostupný na adrese v tvare https://tvoje-meno.github.io/mail-navody/.

Ak chceš, tento základ môžeš ďalej rozšíriť o logo firmy, kontaktné údaje alebo vlastné farby.

Statická stránka pripravená pre GitHub Pages. Súbory zostávajú vo svojich pôvodných formátoch, takže PDF návody sa dajú jednoducho otvoriť alebo stiahnuť.
